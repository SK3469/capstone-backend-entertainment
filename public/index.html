Entertainment App API Documentation

User Routes

POST /user/register

Registers a new user.

POST /user/login

Logs in an existing user.

GET /user/logout

Logs out the current user.

GET /user/profile

Retrieves the authenticated user profile.

GET /user/:id

Gets a user by ID (Requires authentication).

Media Routes

GET /media/trending/:page

Gets trending media for the given page.

GET /media/movie/:page

Retrieves a list of movies.

GET /media/tv/:page

Retrieves a list of TV shows.

GET /media/movie/image/:movieId

Gets the thumbnail image for a movie.

GET /media/tv/image/:seriesId

Gets the thumbnail image for a TV show.

GET /media/movie/detail/:movieId

Retrieves detailed information about a movie.

GET /media/tv/detail/:seriesId

Retrieves detailed information about a TV show.

GET /media/movie/search/:searchQuery

Searches for movies.

GET /media/tv/search/:searchQuery

Searches for TV shows.

GET /media/all/search/:searchQuery

Searches for all media types.

Bookmark Routes

GET /media/bookmark/check/:id

Checks if a media item is bookmarked.

POST /media/bookmark/add

Adds a media item to bookmarks.

DELETE /media/bookmark/delete/:id

Deletes a bookmarked media item.

GET /media/bookmark/get

Retrieves all bookmarked media items.

GET /media/bookmark/search/:searchQuery

Searches for bookmarked media items.
